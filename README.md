- GPU acceleration (CUDA)

- Interactive visualization with Chart.js

## 🧠 Tech Stack
- FastAPI
- PyTorch (GPU)
- Pandas
- Chart.js

## 🔧 Run Locally

pip install -r requirements.txt
uvicorn main:app --reload
